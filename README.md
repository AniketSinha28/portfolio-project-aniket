# Portfolio Project — Aniket Sinha

## Research Topic
**AI-Powered SEO Content Production for B2B SaaS**

## Why This Topic
Search is changing faster than most marketers realise. 
Traditional keyword-led content is being replaced by 
AI-assisted content systems that build topical authority 
at scale. For B2B SaaS companies this represents both 
a threat and an opportunity — the brands that build 
the right content infrastructure now will dominate 
organic acquisition for the next decade.

I chose this topic because it sits at the intersection 
of my performance marketing background and where 
search is genuinely heading. Having built email and 
paid media systems that compound over time, I wanted 
to understand how the same compounding logic applies 
to AI-visible content production.
